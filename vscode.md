# HTML
## 選択範囲をタグで囲う
コマンドパレットで「Emmet Wrap with Attbreviation」を実行

## カレントカーソルのタグを変更する

開始タグと終了タグを両方変更できる
コマンドパレットで「Emmet Update Tag」を実行

## Emmet補完で開始タグと終了タグを一緒に入力する

setting.jsonで以下の設定をしておく

```json
emmet.triggerExpansionOnTab : true
```

タグ名を入力し、tabを入力すると、保管できるようになる

# Markdownプレビュー
`Ctrl` + `Shift` + `V` でプレビュータブを表示。タブなので、編集時は編集しているエディタタブに移動したり、タブを閉じれば、編集を最大サイズで再開できる。
